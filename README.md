# ScheduleGenerator-App
iOS projekt aplikacji pozwalającej obsłużyć algorytm programu ScheduleReader

ContentView
├── FileSekcja
│   ├── Wybór pliku
│   ├── Pokazanie nazwy pliku
│   ├── Wybór nazwy miesiąca
│   └── Button Generowania
└── Sekcja Ustawien
    ├── Edycja listy ludzi
    │   ├── Lista ludzi
    │   │   └── Usuwanie pozycji
    │   └── Dodawanie pozycji
    └── Edycja listy błędów
        ├── Lista błędów
        │   └── Usuwanie pozycji
        └── Dodawnie pozycji
